## How to Use

### Basic Workflow

1. Switch to **Edit Mode**
2. Select the edges you want to bridge
3. **Right-click** to open the context menu
4. Select **Quad Bridge**

---

## Installation

1. Download the addon from:
   - [BlenderKit](https://www.blenderkit.com/addons/8da31723-08d8-48bc-803d-550657841ee8/) (Recommended)
   - [GitHub Releases](https://github.com/abd-3d/quad_bridge/releases)
   - [Gumroad](https://abd3d.gumroad.com/l/quad_bridge)
2. In Blender, go to `Edit > Preferences > Add-ons`
3. Click `Install...` and select the downloaded `.zip` file
4. Enable the addon by checking the box next to "Quad Bridge"
